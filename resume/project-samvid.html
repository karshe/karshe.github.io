Project Title

Mobile website for college event.

Technology used

Frontend HTML5

JS framework jQuery

Cloud GAS



Visit Site

Objective

—
Our college was organizing technical cum cultural level event—to attract numbers of student they went with heavy desktop website, my task was to develop mobile first lightweight website to get ‒more of student as participants.

How we do it

Phase I
PROPER SITEMAP

In order to develop mod of website already existing my half of the task was done already—still I need to prepare proper sitemap for mobile website. Faster loading, proper navigation, student engagement was one of the my first priorities. I prepared a rough sitemap and used Bootstrap to setup rough template for site.

Phase II
UI & UX

Sitemap really helped me while I was planning template for site—I used mobile first strategy, hence I need to proceed with less UI components but still need to provide subtle experience. I used jQuery mobile framework to provide content first website.

Phase III
COMPLETE EXPERIENCE

As soon as website got its shape we were developing experience side by side. User registration was big deal with desktop browser—I need to ease that process in order to achieve our primary goal. I used Google App Script and embedded registration form with custom notifications and triggers.
